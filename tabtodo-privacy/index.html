TabTodo Privacy Policy

Last Updated: January 11, 2024

TabTodo takes your privacy seriously. This privacy policy explains how we handle your data.

Data Collection and Usage

TabTodo only stores todo list data locally in your browser, including:

  • Todo item content
  • Completion status
  • Creation time
  • Sorting information

This data is used solely for displaying, syncing, and sorting your todo items.

Data Storage Location

  • Primary storage: Chrome browser's sync storage
  • Backup storage: Browser's local storage

All data is stored only on your devices; we do not upload any data to external servers.

Data Synchronization

Through Chrome's sync storage, your todo items can be synchronized across devices logged into the same Chrome account. This synchronization process is handled by the Chrome browser, ensuring data security.

Data Security

We protect your data through the following measures:

  • Local-only data storage
  • Use of Chrome's secure storage mechanisms
  • No data sharing with third parties

User Rights

You can at any time:

  • View all your todo item data
  • Delete individual or all todo items
  • Clear extension data through Chrome's settings

Permission Details

TabTodo only requests these essential permissions:

  • storage: for storing todo items
  • tabs: for new tab page functionality
